# HTML

## HTML Introduction

- HTML : Hyper Text Markup Language

  - Hyper Text : 하이퍼링크로 연결된 웹 문서 => 웹 문서(페이지)
  - Markup Language : 표시 언어

- HTML 표시 내용

  - Web Page의 Contents
  - Web page의 Structure

- 기본 구조

※ ` : backtick

```HTML
<!DOCTYPE html>
<html>
  <head>
    웹 문서의 부가정보
  </head>

  <body>
    웹 문서의 내용
  </body>
</html>
```
